# 🚀 Caroline Ribeiro Balles | Portfólio de Tecnologia & IA

Este é o repositório do meu portfólio profissional, desenvolvido para demonstrar a convergência entre **Engenharia de Software**, **IA Generativa** e **Performance de Negócios**.

## 🌐 Visualize Online
[🔗 https://carolineballes.vercel.app](https://carolineballes.vercel.app)

## 🛠️ Tech Stack
*   **Framework:** Next.js 16 (App Router)
*   **Estilização:** Tailwind CSS 4
*   **Animações:** Framer Motion
*   **Ícones:** Lucide React

## 🏗️ Estrutura do Projeto
O site foi construído seguindo os mais altos padrões de arquitetura web:
*   **Hero Section:** Landing page com foco em autoridade e copywriting estratégico.
*   **Cases de Sucesso:** Demonstração de SaaS de Streaming, Agentes de IA e Automações.
*   **Micro-interações:** Botões dinâmicos e componentes com efeito Glassmorphism.
*   **Conversão:** Botão flutuante de WhatsApp para contato direto.

## 📂 Como rodar localmente
1. Clone o repositório:
   `git clone https://github.com/CarolineBalles/caroline-portfolio.git`
2. Instale as dependências:
   `npm install`
3. Inicie o servidor de desenvolvimento:
   `npm run dev`
